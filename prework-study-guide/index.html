HTML

  • Enter your HTML notes here
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • A semantic element clearly describes its meaning to both the browser and the developer eg the form and table elements.
  • Element and tag are used interchangably, however, elements refer to the entire element, from from opening tag to closing tag while tag refers only to what appears inside the angle bracket.
  • The header tag appears at the top of a page and usually contains a navigation menu.
  • When using headings, it is important to use them in order from h1 to h6.
  • The biggest heading h1 should only appear once on each page.
  • HTML uses the id attribute as a way to identify elements and select them.

CSS

  • Enter your CSS notes here
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around thecontent inside an element.
  • A selector defines the element or attributes to which the rules or declearations will apply
  • Declearations contain two important components: the css property and the value.
  • display: block; asssigns the element to start on a new line.
  • Class attributs allows us to share a CSS rule to any element we choose by assigning the rule to the class attribute with a class selector.
  • The box-shadow property adds a shadow effect around an elements frame.

Git

  • Enter your Git notes here
  • git status: Checks which branch you're currently on
  • git checkout -b branch-name: Creates a new branch and switches to it
  • git checkout -b feature/add-html: Creates a new branch named feature/add-html
  • git clone: Clones a repo into a new directory
  • git add: Adds file contents to index
  • git commit: Records changes to the repository
  • git branch: Lists, creates or deletes branches
  • git pull: Fetches and downloads content from a remote repository and immediately updates the local repo to match that content
  • git push: Uploads local repository content to a remote repository
