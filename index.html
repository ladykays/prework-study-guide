HTML

  • Enter your HTML notes here
  • HTML means Hypertext Markup Language
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • A semantic element clearly describes its meaning to both the browser and the developer eg the form and table elements.
  • Element and tag are used interchangably, however, elements refer to the entire element, from from opening tag to closing tag while tag refers only to what appears inside the angle bracket.
  • The header tag appears at the top of a page and usually contains a navigation menu.
  • When using headings, it is important to use them in order from h1 to h6.
  • The biggest heading h1 should only appear once on each page.
  • HTML uses the id attribute as a way to identify elements and select them.

CSS

  • Enter your CSS notes here
  • CSS means Cascading Style Sheet
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around thecontent inside an element.
  • A selector defines the element or attributes to which the rules or declearations will apply
  • Declearations contain two important components: the css property and the value.
  • display: block; asssigns the element to start on a new line.
  • Class attributs allows us to share a CSS rule to any element we choose by assigning the rule to the class attribute with a class selector.
  • The box-shadow property adds a shadow effect around an elements frame.
  • The CSS file is linked within the head element usint the link tag

Git

  • Enter your Git notes here
  • git status: Checks which branch you're currently on
  • git checkout main: To switch to main branch
  • git
  • git checkout -b branch-name: Creates a new branch and switches to it
  • git checkout -b feature/add-html: Creates a new feature branch named feature/add-html
  • git clone: Clones a repo into a new directory
  • git add: Adds file contents to index
  • git add -A: To add all modifications in the staging area
  • git commit: Records changes to the repository
  • git commit -m "description of what was done": To commit changes with comment describing what was done
  • git branch: Lists, creates or deletes branches
  • git pull: Fetches and downloads content from a remote repository and immediately updates the local repo to match that content
  • git pull origin main: Pulls the remote main branch into the local main branch so local and remote are in sync
  • git push: Uploads local repository content to a remote repository
  • touch script.js: creates a new file named script.js

JavaScript

  • Enter your JavaScript note here
  • JavaScript is an Object Oriented Programming (OOP) language.
  • To declare a variable, use the var keyword
  • After the var keyword, give the variable a unique name
  • Finally, use the assignment operator, = , to assign the specific data that is to be stored in the variable
  • var shape = "circle"; Shape is the variable and the data it stores is "circle"
  • All strings must be contained within quotation marks " "
  • The console.log method is used to output a message to the web console
  • The script file is linked within the body element using the script tag
  • var topics = "HTML, CSS, Git, JavaScript";: This is a variable declearation with string values
  • console.log(topics); : Logs variables to the web console in browser
  • Control flow is the order in which a computer executes code in a script.
  • Computers read code in a linear way (from top to bottom)
  • Developers can achieve the logic of causing a computer to deviate from the normal linear flow by using if statements.
  • An array is a single value that is used to hold a group of related data
  • To declare an array, you use the var keyword eg var shapes = ["circle", "square", "oval", "triangle"];
  • Arrays are useful when performing a task on multiple pieces of data at once.
  • A single array can store differnt data types such as string, number, boolean etc.
  • Arrays are zero-indexed and sequential i.e the first item in an array has an index of 0, not 1
  • In order to access an individual item in an array, we use an array index
  • An array index is a number indicating each items place in the order of the list
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing the code block to be executed over and over.
  • A function is simply a set of instructions that tells the computer how to perform a certain task.
  • A function is different from a for loop or conditional statement because functions do not execute automatically when a JS file is run. A function must be called by name in order to execute it.
  • A function groups a set of instructions so each time you need it, all you need to do is call the function.
  • Functions in JS are used to define and reuse blocks of code.
  • It is important to note that a function name is followed by parenthesis.
  • A function must have a unique name.
